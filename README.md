**Project Name:-**

**Markio**

**Description**
Markio is a group project built using **HTML, CSS, and JavaScript.**

It is an **automated student attendance monitoring and analytics system for colleges**, enabling secure, location-based attendance marking with cloud-based tracking and insights for students, faculty, and administrators.

Features

1.**UUID-based student login** with teacher session code verification<br>
2.**Geo-fencing to ensure attendance** is marked only within campus/classroom boundaries<br>
3.Offline and online attendance support with **automatic data sync**<br>
4.Misuse detection with **warnings and penalty tracking**<br>
5.**Cloud-based dashboards** with attendance analytics<br>
